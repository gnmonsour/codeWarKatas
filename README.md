## Naming Conventions
- Class names s/b almost always `PascalCase` convention

- Identifiers in `C/C++`, `Python`, `Ruby` follow `snake_case`

- Identifiers in `Java`, `JavaScript`, `Haskell` follow `camelCase`

- Identifiers in `C#` are *always* `PascalCase`
